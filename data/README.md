# Dataset availability

The dataset is not included in this repository.

The hybrid dataset used in the associated study combines:

1. newly generated experimental results obtained in the present research; and
2. laboratory data underlying the Flexible Pavement Design Guide published by the General Directorate of Highways (KGM, 2008).

Although the guide is publicly available, the underlying KGM laboratory database is not publicly distributed and was accessed with permission exclusively for scientific research purposes within the scope of the R&D project acknowledged in the manuscript.

Therefore, the complete hybrid dataset cannot be redistributed through GitHub or as supplementary material.

The experimental results generated in the present study are reported in the associated manuscript.
